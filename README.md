# О проекте
Это первое задание тестового курса техношколы WB, в нем было необходимо разработать демонстрационный сервис с простейшим интерфейсом, отображающий данные о заказе. Он получает заказы из топика Kafka, валидирует их, сохраняет в базу данных PostgreSQL и кэширует в LRU-кэше для быстрого доступа.
